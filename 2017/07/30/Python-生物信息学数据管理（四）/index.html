break和continue语句

  • 当解释器碰到break语句时会退出循环，不再执行循环语句的其余部分，包括else语句（for循环中的else语句）

  • continue语句跳过循环语句的其余部分，跳至下一次循环

  • startswith可以用来匹配字符串开头是否包含一个字符串

  • ‘endswith’用来匹配字符串末尾是否包含一个字符串
