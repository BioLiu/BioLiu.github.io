模式匹配

检验正则表达式是否正确的在线工具
search()函数扫描字符串并寻找正则表达式第一次匹配的位置，其返回的每个序列最多只能有一个单一匹配对象。group()方法打印第一个匹配的对象。

search()方法返回Match object而并非直接返回匹配的字符串，其中编码了匹配的子字符串，以及开始和结束的位置

  • match.group() 返回匹配字符串
  • match.span() 返回匹配结果的包含起点和终点位置的元组
  • match.start() 返回匹配结果的起始位置
  • match.start() 返回匹配结果的终止位置

若只从序列的第一位置寻找正则表达式匹配，则使用match()方法

search()及match()方法返回匹配对象，该对象可以通过匹配对象方法传递给变量

我们也可以搜索所有匹配

  • findall() 返回包含所有匹配的一个字符串列表
  • finditer() 以迭代器的形式返回，可以通过for遍历

分组

将一个正则表达式分解为若干组

group()方法不填写参数或参数等于0，返回完全匹配的子字符串；子组以从左到右编号，向右递增编号(从1开始)

向group()传递多个参数，将返回包含各组的值对应的元组

groups()返回所有与子组相关的子字符串

还可以向每个子组分配名称即?P<名>，以便选择性的检索其内容即group(<名>)

  • sub(r,s) 找到所有RE匹配的子串，替换成不同的子串

字符和元字符

  • [^a] 除了a以外的字符
  • ^[a] a只匹配s的首位
  • () 在正则表达式中创建子组
  • * 匹配零次或多次
  • + 匹配一次或者多次
