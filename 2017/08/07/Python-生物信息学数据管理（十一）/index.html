按第二列对表排序

从文本文件中读取Lowry数据表，按照第二列排序，将已经排序表的前三行写入新文件
Lowry数据表内容

实现代码如下：
import os
from operator import itemgetter
os.chdir(“F:/Python_biologist”)
input_file = open(“lowry_data.txt”)
output_file = open(“lowry_dataSortedThird.txt”,”w”)

#将文件数据读取到列表中
table = []
header = input_file.readline()
for line in input_file:
col = line.strip().split(‘\t’)
col[1] = float(col[1])
table.append(col)

#排序
table_sorted = sorted(table,key=itemgetter(1))

#写入到输出文件
output_file.write(header)
for row in table_sorted[0:3]:
row = [str(x) for x in row]
output_file.write(“\t”.join(row)+”\n”)
output_file.close()
