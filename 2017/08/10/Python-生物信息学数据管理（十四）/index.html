命令的含义

  • get_residues(pdb_file)读取PDB输入文件，提取氨基酸三字码和链型，将每个氨基酸残基组成[res_type,chain]储存在名为residues的Python列表中。通过名称’CA’(if ca == ‘CA’)确保每个残基只使用一次
  • threeletter2oneletter(residues)将氨基酸三字码转换为单字码，内置函数enumerate(residues)生成(n,residues[n])形式的元组，此处n为从零计数的整数
  • write_fasta_records(residues,pdb_id,fasta_file)函数将残基列表格式化为FASTA格式的字符串

带有range()和xrange()的循环

  • range(n,m)创建从n到m-1的整数
  • xrange(n,m)用于创建迭代器，xrange()仅在需要时生成与对应列表相同的值，占内存更小，可用于很大的数值

必选参数

一个或多个参数必须传递给函数，参数在调用中的顺序必须与函数定义中的顺序完全一致

关键字参数

给函数的参数分配一个名称，可以忽略顺序

默认参数(可选参数)

  • 默认参数为不可变参数数字，字符串
  • 默认参数，必须放在函数定义的最后位置

变长参数

参数的个数是可变的(即函数的一个调用和另一个调用的参数个数是不同的)

  • 对于元组参数符号*
  • 对于字典参数符号**
