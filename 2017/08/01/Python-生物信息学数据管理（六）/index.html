集合

在上述的很多例子中，已经见到了集合的强大功能，接下来，我们将对其其他功能进行介绍
集合的定义：唯一对象的无序组合
集合是删除重复、计算交集、并集以及两个或以上对象组之间差异的理想数据结构。集合支持in和not in逻辑计算

创建集合

集合中的元素为不可变对象，因此列表、字典或者其他集合不能作为集合的元素
由于集合是唯一元素的组合，因此在创建集合时，会自动将重复的的元素删除

集合的方法

add()将一个元素添加到集合

update()方法将多个元素添加到集合

pop()，remve()及discard()方法可以从集合中去除元素

内置函数reduce()有两个参数：

  • 有两个变量的函数(f(x,y))
  • 迭代对象i（元组或列表）

reduce(f,i)将两个迭代元素（i[0]和i[1]）传递至函数f，计算f返回的值，然后将该值作为f的第一个参数，迭代器的第三个元素（i[2]）作为参数，以此类推。
其中f也可以为自定义函数
